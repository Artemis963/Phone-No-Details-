    <h1>Phone Number Details App</h1>

   <p>The Phone Number Details App is a Flutter-based mobile application that allows users to input a phone number and receive details such as the country code and validity status of the phone number. This simple and efficient app provides a user-friendly interface for quick and accurate phone number analysis.</p>
    <img src="screenshots/phone_details_app.png" alt="App Screenshot" />
    <h2>Features</h2>
    <ul>
        <li><strong>Phone Number Analysis:</strong> The app provides details about the input phone number, including the country code.</li>
        <li><strong>Validity Check:</strong> It also checks the validity of the phone number and informs the user whether it is valid or not.</li>
        <li><strong>User-Friendly Interface:</strong> The app offers an intuitive and visually appealing user interface for a seamless experience.</li>
    </ul>
    <h2>Getting Started</h2>
    <ol>
        <li>Clone the repository to your local machine:</li>
    </ol>
    <code>git clone https://github.com/sayanth-t-m/Phone-No-Details-.git</code>
    <ol start="2">
        <li>Navigate to the project directory:</li>
    </ol>
    <code>cd Phone-No-Details-</code>
    <ol start="3">
        <li>Install dependencies using Flutter:</li>
    </ol>
    <code>flutter pub get</code>
    <ol start="4">
        <li>Run the app on your preferred emulator or device:</li>
    </ol>
    <code>flutter run</code>

   <h2>Usage</h2>
    <ol>
        <li>Open the Phone Number Details App on your device.</li>
        <li>Input the phone number you want to analyze.</li>
        <li>Press the "Get Details" button.</li>
       <li>The app will display the country code and inform you whether the phone number is valid or not.</li>
</ol>
